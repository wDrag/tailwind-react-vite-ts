# Tailwind + React + TypeScript + Vite

This template provides a minimal setup + file structure to get Tailwind + React + TypeScript working in Vite with HMR and some ESLint rules.

## File Structure

```
.
├── public
│   └── example.svg
├── src
│   ├── App.tsx
│   ├── components
│   │   ├── home
│   │   │   └── index.tsx
│   │   └── shared
│   │       └── index.tsx
│   ├── constants
│   │   ├── env.ts
│   │   └── index.ts
│   ├── index.css
│   ├── index.html
│   ├── main.tsx
│   ├── pages
│   │   └── Home.tsx
│   ├── Router.tsx
│   ├── types
│   │   └── index.ts
│   └── utils
│       └── index.ts
├── vite.config.ts
└── package.json
```

## Usage

use `npm`, `yarn` or `pnpm`, etc.

### Install Dependencies

```bash
npm install
```

### Run

```bash
npm run dev
```

### Build

```bash
npm run build
```
